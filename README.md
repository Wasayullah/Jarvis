# 🤖 Jarvis-549 AI Assistant

Jarvis-549 is an advanced Python-based intelligent AI assistant designed and developed by **Mohammad Wasayullah**.  
It operates as a hybrid system combining rule-based logic with a local LLM (Ollama Qwen model), memory persistence, and system automation.

---

## 🚀 Features

- 🧠 Hybrid AI system (Rules + Local LLM via Ollama)
- 💬 Conversational memory (stores chat history per user)
- 🖥️ System automation (open apps like Chrome, VS Code, CMD, Notepad)
- 🌐 Web operations (Google, YouTube, websites)
- 📖 Wikipedia search integration
- 🌦️ Weather API integration
- 🎵 YouTube & Spotify playback support
- 📸 Screenshot capture
- 🧾 Personalized AI behavior (creator vs user mode)
- 💾 JSON-based persistent memory system
